## About Cdev+

Cdev+ is a free, open source mobile companion for Claude Code & Codex that enables you to:

- Monitor Claude Code and Codex sessions in real-time
- View terminal output and logs remotely
- Send prompts and interact with your agent using text or voice
- Browse workspace files and view diffs
- Monitor source control status
- Manage tool permissions
- Connect via QR code pairing

**Requirement:** Cdev+ requires a [cdev](https://github.com/brianly1003/cdev) server reachable from your device (local network or remote host you configure). The app does not function independently.
